About Marauroa Server Manager

Marauroa Server Manager is a manger for your Marauroa based applications. Here are the tasks that can be done with it:
  • Start/Stop your server
  • Visually display the contents of the application (Zones and Objects within) Feature not implemented yet
  • Add, remove and/or modify RPObjects on real time. You can be a Dungeon Master on real time. Feature not implemented yet
  • For supported applications, have a world wide view of the server, like a Google map with real time location of objects in the world. Feature not implemented yet
See: https://sourceforge.net/apps/mediawiki/simple-marauroa/index.php?title=RCP_Server_Manager for more details.
